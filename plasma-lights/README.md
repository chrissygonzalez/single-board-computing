# Lights

## The Problem

Strings of lights are festive, but I don't have space to store separate ones for every season. There's got to be a better way!

## Hardware Used

- [Pimoroni Plasma 2040](https://shop.pimoroni.com/products/plasma-2040)
- [RGB LED wire](https://shop.pimoroni.com/products/rgb-led-wire)
